# l6_capstone
Repo for Capstone (dissertation) module for the BASc in Iterdisciplinary Problems and Methods at the London Interdisciplinary School (LIS)

===SUPPLEMENTARY MATERIAL FOR THE ACADEMIC OUTPUT===

Before running the notebook, you will need to download the bills, laws, and speeches of Denmark, Germany and Austria from https://parllawspeech.org/ and place the RDS files in their appropriate folders within the data directory.

You will need your own Claude API key, known to the Python environment e.g. using s.environ['ANTHROPIC_API_KEY'] = ..., and credits on your account to regenerate the AI's document classification. You will also need to delete or rename the existing citation_classifications.csv to do this.
